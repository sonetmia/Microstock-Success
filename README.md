# Microstock Success - Complete AI-Powered Toolkit

A high-performance web application designed for microstock contributors to streamline their production workflow. Built with **React**, **Vite**, **Tailwind CSS 4**, and **Groq AI**.

## 🚀 Features

- **Microstock Journey With Sonet**: A 9-step proven strategy for building a successful portfolio.
- **AI Prompt Generator**: Optimize your image generation prompts for high-converting stock aesthetics.
- **AI Metadata Forge**: Generate SEO-optimized titles, descriptions, and 30+ keywords using Vision AI (Llama 3.2).
- **Resource Nexus**: Instant access to secondary tools (Upscalers, Background Removers, etc.).
- **Client-Side Security**: Your Groq API key is stored only in your browser (LocalStorage).

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite 6
- **Styling**: Tailwind CSS 4 + shadcn/ui Inspired components
- **Navigation**: React Router 7
- **AI Engine**: Groq Cloud SDK
- **Animation**: Motion (framer-motion)
- **Icons**: Lucide React

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd microstock-success
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🔑 AI Setup

This application requires a **Groq API Key** to function.
1. Visit [Groq Cloud Console](https://console.groq.com/keys) to create a free API key.
2. Navigate to the **Settings** page in the app.
3. Paste your key and save.
